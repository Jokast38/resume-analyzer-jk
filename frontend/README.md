
# Frontend Resume Reader

Ce dossier contient l’interface web du projet Resume Reader, développée avec React et Tailwind CSS.

## Scripts disponibles

Dans ce dossier, vous pouvez lancer :

- `npm start` : démarre l’application en mode développement ([http://localhost:3000](http://localhost:3000))
- `npm test` : lance les tests unitaires
- `npm run build` : construit l’application pour la production

## Documentation complète

Consultez le fichier [README.md](../README.md) à la racine du projet pour la documentation globale, l’architecture, l’installation et l’utilisation du backend et du frontend.
